Common questions

My subscription didn't unlock features

Open Settings (gear icon, top-right of any tab) → Restore Purchases. Make sure you're signed in to the same Apple ID you used to buy.

How do I cancel my subscription?

iOS Settings → Apple ID (top of Settings) → Subscriptions → CleanCut Pro → Cancel. Cancel at least 24 hours before the end of the period to avoid the next charge. The 7-day free trial converts automatically if not cancelled before it ends.

Smart Cut isn't detecting my filler words

Make sure you've set the right recognition language for the session (the language pill at the top of the Smart Cut studio). Smart Cut works best on clear speech recorded close to the microphone. For very noisy recordings, run Denoise first, then export and run Smart Cut on the cleaned file.

Where are my exported files?

Use the Share sheet from the export screen — Save to Files puts the file in any folder you choose (including iCloud Drive). The Share Extension makes the file available to any app that accepts audio (Voice Memos, Mail, Telegram, etc.).

Can I get a refund?

Refunds are handled by Apple, not by us. Request one at reportaproblem.apple.com.
